# 테스트 시나리오

- 사레 관점 
    - cb, 이벤트의 호출 흐름
        - baseXXX, cbCmd
    - output 옵션별 : 0, 1, 2, 3
    - outPut 추가
    - bind 구성방식
        1. 아이템 추가 후 커맨드 설정
        2. 컬럼 추가 및 커맨드 동시 등록
        3. 컬렉션에 컬럼 추가 후 컬럼 설정
        4. 커멘드에 컬럼 직접 추가 (단일 설정에 적합)
        5. 서비스 등록
    - 커멘드에 기본테이블 변경 후 추가
    - preXXX 흐름
    - 에러와 실패
    - 제약 조건
    - valid() 와 컬럼


- 구조 관점 : 속성, 메소드, 이벤트
    - 이벤트 전파 금지
